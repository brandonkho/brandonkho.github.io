projects

FoodMate:

FoodMate is a web app that uses Ruby on Rails and the Yelp API to recommend restaurants to users based on their food preferences. Users can log in and the app will recommend restaurants in the area depending on the users zip code and match the user with another user so they can eat together.

Network:

A machine player in Java that plays the game Network against human and computer players. It uses game tree search and alpha-beta pruning to choose the most optimal moves. Depth-first search is used to check if there is a winning solution.

Firewall:

I implemented a firewall in Python that enforced a set of rules defined by the user. Some features included dropping TCP packets and denying DNS packets by injecting RST packets with a set DNS.
